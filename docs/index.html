Privacy Policy

This document is the privacy policy for the MigStatus and CaseStatus-application.


Data collection

The application stores

  • The application number and/or application date you provide
  • The answers to the questions in the time to a decision you provide (waiting time answers)

The application uses AdMob and Firebase by Google, these may use personal information
such as usage of applications to determine your interests in order to provide targeted ads.
If you want more information about Google Ads please visit this link
If you want to learn more about the information Google may store about you visit this link
Firebase also stores some information about the application usage in order to provide better analytics of the usage of the application.

Data usage

The information you provide (application number, answers to time to a decision questions) are only
used to query the migrationsverket website.
The communication with the migrationsverket website is done using HTTPS.

The information Google's firebase and/or AdMob may collect is used to provide targeted ads and/or
analytics of user behaviour, such as average daily time spent in the app and usage of the application in different countries.

Opt out

If you want to remove the information you have provided (application number/date and waiting time answers)
select the 'reset application' option in the menu in the application.
The application may be used using only the date of the application and a chosen average time to a decision.
